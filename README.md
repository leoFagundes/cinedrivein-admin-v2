## Sobre

Painel utilizado pela equipe para gerenciar pedidos, estoque, usuários e a configuração do site público. Acesso restrito a usuários autorizados.

## Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS
- Firebase

## Instalação

```bash
npm install
npm run dev
```
